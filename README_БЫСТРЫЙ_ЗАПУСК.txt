Электронный акт обследования РСОН v0.12.1

Ссылка после публикации GitHub Pages:
https://torconsult-dev.github.io/4_act_rson-pwa/

Что изменено в v0.12.1:

- добавлен контроль обязательного заполнения перед отправкой и формированием PDF;
- если не заполнен хотя бы один обязательный пункт, акт не отправляется и PDF не формируется;
- на экране «Подписи и отправка» показывается список незаполненных пунктов;
- сохранён единый зелёный стиль всех элементов выбора из v0.12;
- сохранены GPS, населённый пункт по GPS, фото, пояснения к фото, голосовая надиктовка, эскиз, ластик, отмена действия, подписи, QR, индивидуальный номер объекта и PDF.

Для обновления GitHub Pages загрузить все файлы из архива в корень репозитория с заменой и нажать Commit changes.
После обновления на телефоне закрыть старую вкладку или очистить кэш, чтобы не подтягивалась старая версия.
