Как обновить GitHub Pages:
1. Откройте репозиторий rson-pwa на GitHub.
2. Загрузите файлы из этой папки в корень репозитория с заменой существующих:
   index.html
   Паспорт_объекта_app.html
   manifest.webmanifest
   service-worker.js
   icon-192.png
   icon-512.png
3. Нажмите Commit changes.
4. Подождите 1-3 минуты и обновите ссылку:
   https://newbiejava.github.io/rson-pwa/

Важно:
Полный акт с фото и подписями формируется через PDF.
Кнопка «Отправить отчет» передает текстовую часть письмом и не заменяет полный PDF-акт.
