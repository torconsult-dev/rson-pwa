БЫСТРЫЙ ЗАПУСК «ПАСПОРТ ОБЪЕКТА РСОН»

Файлы:
- index.html — основной файл для размещения на сайте/хостинге.
- Паспорт_объекта_app.html — исходное имя файла, оставлено для совместимости.
- manifest.webmanifest — файл установки PWA.
- service-worker.js — офлайн-кэширование.
- icon-192.png / icon-512.png — иконки приложения.

ВАРИАНТ 1. Быстро открыть на этом компьютере
1. Откройте папку с файлами.
2. Запустите run_local_server.bat.
3. Откройте в браузере:
   http://localhost:8000/

ВАРИАНТ 2. Быстро открыть с телефона в той же Wi-Fi сети
1. Запустите run_local_server.bat на компьютере.
2. Узнайте IP компьютера:
   ipconfig
3. На телефоне откройте:
   http://IP_КОМПЬЮТЕРА:8000/
   Например:
   http://192.168.1.25:8000/

Важно:
- Через обычный http на телефоне камера обычно работает, но GPS и микрофон могут работать нестабильно.
- Для нормальной работы GPS, микрофона и установки на главный экран нужен HTTPS.

ВАРИАНТ 3. Нормальный запуск с любого устройства
Разместите всю эту папку на HTTPS-хостинге:
- Netlify Drop
- Cloudflare Pages
- GitHub Pages
- корпоративный сайт/сервер с HTTPS

После публикации откройте выданную ссылку на Android/iPhone и добавьте на главный экран.

Android:
Chrome → меню ⋮ → Добавить на главный экран.

iPhone:
Safari → Поделиться → На экран «Домой».
