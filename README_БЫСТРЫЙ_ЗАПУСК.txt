Электронный акт предпроектного осмотра объекта v0.2.7

PWA-приложение для заполнения одного электронного акта предпроектного осмотра объекта по обновлённой форме.

Что исправлено в v0.2.7:
- кнопка «Новый акт» теперь не перезагружает страницу и не даёт старому черновику сохраниться обратно через beforeunload;
- очистка удаляет черновики текущей и старых версий, затем создаёт новый пустой акт с новым номером;
- кнопка «Продолжить черновик» ищет черновик v0.2.7 и предыдущих версий, открывает сохранённую страницу и сохранённый блок;
- добавлена защита от повреждённого/несовместимого черновика;
- блок ошибки обязательных полей теперь показывается на текущей активной странице, а не на скрытой странице;
- под полями ввода добавлены кнопки «Надиктовать», «Стоп запись», «Очистить», включая числовые поля кабельных трасс и уровней сигнала;
- для одиночного и множественного выбора сохранена кнопка очистки выбора;
- обновлён service-worker и имя кэша, чтобы браузер не держал старую v0.2.5.

Как запустить:
1. Распакуйте архив.
2. Откройте index.html в браузере или загрузите все файлы в корень GitHub Pages.
3. Для PWA/мобильного режима откройте через https-адрес GitHub Pages и добавьте приложение на главный экран.

Если после обновления на телефоне всё ещё открывается старая версия:
- закройте вкладку приложения;
- очистите данные сайта / кэш браузера для адреса приложения;
- откройте страницу заново.
