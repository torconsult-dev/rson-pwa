Электронный акт предпроектного осмотра объекта v0.2.5

PWA-приложение для заполнения одного электронного акта предпроектного осмотра объекта по обновленной форме.

Что изменено в v0.2.5:
- убрано фиксированное название акта;
- название акта вынесено в отдельное заполняемое поле на первой странице;
- поле «Название акта» можно заполнить вручную или надиктовать голосом;
- QR-код и PDF используют заполненное название акта.

Для запуска через GitHub Pages загрузите файлы архива в корень репозитория и включите Pages: main / root.


Репозиторий с кодом:
https://github.com/torconsult-dev/rson-pwa

Ссылка GitHub Pages для запуска приложения:
https://torconsult-dev.github.io/rson-pwa/


Версия v0.2.5: для всех полей, где указаны координаты, добавлена кнопка определения по GPS.


Изменения v0.2.5:
- Исправлены кнопки «Отменить» и «Повторить» в эскизе.
- Большинство одиночных выборов сделано выпадающими списками.
- Телефон — только цифры, E-mail — только формат почты.


Версия v0.2.5:
- стартовая страница: Новый акт очищает черновики, Продолжить черновик открывает сохранённые данные;
- блоки формы разделены на отдельные страницы по смыслу;
- переход дальше невозможен, пока не заполнены обязательные поля текущей страницы;
- под текстовыми полями добавлены кнопки: Надиктовать / Стоп запись / Очистить;
- запрещён запуск более одной надиктовки одновременно;
- надиктовка автоматически останавливается через 5 секунд без голосового ввода;
- добавлены комментарии к полям с материалами;
- сохранена проверка телефона, числовых полей, метров и E-mail.
