Электронный акт предпроектного осмотра объекта v0.2.3

PWA-приложение для заполнения одного электронного акта предпроектного осмотра объекта по обновленной форме.

Что изменено в v0.2.3:
- убрано фиксированное название акта;
- название акта вынесено в отдельное заполняемое поле на первой странице;
- поле «Название акта» можно заполнить вручную или надиктовать голосом;
- QR-код и PDF используют заполненное название акта.

Для запуска через GitHub Pages загрузите файлы архива в корень репозитория и включите Pages: main / root.


Репозиторий с кодом:
https://github.com/torconsult-dev/rson-pwa

Ссылка GitHub Pages для запуска приложения:
https://torconsult-dev.github.io/rson-pwa/


Версия v0.2.3: для всех полей, где указаны координаты, добавлена кнопка определения по GPS.


Изменения v0.2.3:
- Исправлены кнопки «Отменить» и «Повторить» в эскизе.
- Большинство одиночных выборов сделано выпадающими списками.
- Телефон — только цифры, E-mail — только формат почты.
